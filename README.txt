
Проект блога на React (десктопное приложение)

Возможности:
    - чтение постов других авторов;
    - создание личного кабинета;
    - оценка постов (*);
    - публикация собственных постов (*).

* - только для зарегистрированных пользователей.

Использованные технологии: react, redux, routing
